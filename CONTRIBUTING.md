# Contributing

This repository is not accepting external contributions at this stage.

## Current Policy

- Do not open pull requests for feature changes, refactors, or documentation updates.
- If you want to adapt the skill for your own workflow, fork the repository or copy the package into your local skills directory and maintain your own variant.
- If you need to report a security issue, use the private process in `SECURITY.md`.

## Maintainer Workflow

Changes in this upstream repository should keep the package lightweight and documentation-first:

- keep install instructions accurate for both Codex and Claude Code
- keep markdown and links passing in CI
- use Git tags for releases
- avoid adding contribution machinery until the project is ready to accept it
